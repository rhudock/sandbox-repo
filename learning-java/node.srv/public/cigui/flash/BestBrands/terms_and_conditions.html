Description of Services

Co-Browse is a technology that allows the sales agent to view your web page with your authorization to provide additional sales assistance. You have the ability to opt out of the co-browse session at any time by doing choosing the exit co-browse option on the display bar that communicates an active Co-browse session. The Co-browse session will also be terminated once you or the sales agent navigates away from the website from which the co-browse session was initiated. No install or plug-ins are required to run Co-Browse.

Privacy Statement

If prior to engaging in the Co-browse session, you entered personal information, including but not limited to name, email address, birth date, gender, ZIP code, address, Social Security number, occupation, industry, personal interests, phone number, or information that is not otherwise publicly available, all of this personal information will be masked and not visible to the sales agent during the Co-Browse session.
